QShortcut* GetHotkey ( const QString &  group,
const QString &  action,
QWidget *  widget 
)

Returns a QShortcut object whose activated() signal can be connected to other QObjects' slots.

Parameters
groupGeneral group this hotkey belongs to (e.g. "Main Window", "Debugger").
actionName of the action (e.g. "Start Emulation", "Load Image").
widgetParent widget of the returned QShortcut.
Warning
If multiple QWidgets' call this function for the same action, the returned QShortcut will be the same. Thus, you shouldn't rely on the caller really being the QShortcut's parent.
