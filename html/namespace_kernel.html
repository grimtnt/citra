◆ Kernel::Session

class Kernel::Session

Parent structure to link the client and server endpoints of a session with their associated client port.

The client port need not exist, as is the case for portless sessions like the FS File and Directory sessions. When one of the endpoints of a session is destroyed, its corresponding field in this structure will be set to nullptr.
