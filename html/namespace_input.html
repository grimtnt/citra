◆ MotionDevice

using Input::MotionDevice = typedef InputDevice<std::tuple<Math::Vec3<float>, Math::Vec3<float> >>

A motion device is an input device that returns a tuple of accelerometer state vector and gyroscope state vector.

For both vectors: x+ is the same direction as LEFT on D-pad. y+ is normal to the touch screen, pointing outward. z+ is the same direction as UP on D-pad.

For accelerometer state vector Units: g (gravitational acceleration)

For gyroscope state vector: Orientation is determined by right-hand rule. Units: deg/sec

◆ TouchDevice

using Input::TouchDevice = typedef InputDevice<std::tuple<float, float, bool> >

A touch device is an input device that returns a tuple of two floats and a bool.

The floats are x and y coordinates in the range 0.0 - 1.0, and the bool indicates whether it is pressed.
