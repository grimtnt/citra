◆ VerifyStringTableLength()

static ResultCode Service::LDR::VerifyStringTableLength ( VAddr  address,
u32  size 
)
static

Verifies a string or a string table matching a predicted size (i.e.

terminated by 0) if it is not empty. There can be many other nulls in the string table because they are composed by many sub strings. This function is to check whether the whole string (table) is terminated properly, despite that it is not actually one string.

Parameters
addressthe virtual address of the string (table)
sizethe size of the string (table), including the terminating 0
Returns
ResultCode RESULT_SUCCESS if the size matches, otherwise error code.
