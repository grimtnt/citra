◆ PipeRead()

std::vector< u8 > AudioCore::DspHle::PipeRead ( DspPipe  pipe_number,
u32  length 
)

Reads length bytes from the DSP pipe identified with pipe_number.

Note
Can read up to the maximum value of a u16 in bytes (65,535).
IF an error is encoutered with either an invalid pipe_number or length value, an empty vector will be returned.
IF length is set to 0, an empty vector will be returned.
IF length is greater than the amount of data available, this function will only read the available amount.
Parameters
pipe_numbera DspPipe
lengththe number of bytes to read. The max is 65,535 (max of u16).
Returns
a vector of bytes from the specified pipe. On error, will be empty.
