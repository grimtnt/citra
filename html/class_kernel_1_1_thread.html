◆ GetWaitObjectIndex()

s32 Kernel::Thread::GetWaitObjectIndex ( WaitObject *  object) const

Retrieves the index that this particular object occupies in the list of objects that the thread passed to WaitSynchronizationN, starting the search from the last element.

It is used to set the output value of WaitSynchronizationN when the thread is awakened. When a thread wakes up due to an object signal, the kernel will use the index of the last matching object in the wait objects list in case of having multiple instances of the same object in the list.

Parameters
objectObject to query the index of.
