◆ tls_slots

std::vector<std::bitset<8> > Kernel::Process::tls_slots

The Thread Local Storage area is allocated as processes create threads, each TLS area is 0x200 bytes, so one page (0x1000) is split up in 8 parts, and each part holds the TLS for a specific thread. This vector contains which parts are in use for each page as a bitmask. This vector will grow as more pages are allocated for new threads.
