Detailed Description

This is what a client [person joining a server] would use.

It also has to be used if you host a game yourself (You'd create both, a Room and a RoomMembership for yourself)

Enumerator
Uninitialized 

Not initialized.

Default state.

Some error [permissions to network device missing or something].

Joining 

The client is attempting to join a room.

Joined

Connection closed.

The room is not responding to a connection attempt.
