Detailed Description

A helper class parsing and verifying a string-type Path.

Every archives with a sub file system should use this class to parse the path argument and check the status of the file / directory in question on the host file system.

◆ IsValid()

bool FileSys::PathParser::IsValid ( ) const
inline

Checks if the Path is valid.

This function should be called once a PathParser is constructed. A Path is valid if:

  • it is a string path (with type LowPathType::Char or LowPathType::Wchar),
  • it starts with "/" (this seems a hard requirement in real 3DS),
  • it doesn't contain invalid characters, and
  • it doesn't go out of the root directory using "..".
